Политика конфиденциальности

1. Общие положения

Настоящая Политика конфиденциальности определяет порядок сбора, хранения и использования персональных данных пользователей сервиса EventPro.


2. Какие данные мы собираем

Имя и контактные данные (телефон, email)
Данные о мероприятиях, площадках и подрядчиках
Информация о действиях в системе


3. Как мы используем данные

Для предоставления доступа к функционалу платформы
Для связи с вами по вопросам организации мероприятий
Для улучшения качества сервиса


4. Хранение данных

Все данные хранятся в вашем браузере (localStorage) и не передаются третьим лицам. Мы не используем файлы cookie для отслеживания.


5. Права пользователя

Вы можете в любой момент удалить все свои данные через интерфейс платформы или очистив localStorage браузера.


6. Контакты

По вопросам конфиденциальности: privacy@eventpro.ru

Пользовательское соглашение

1. Предмет соглашения

Настоящее соглашение регулирует отношения между EventPro и пользователями платформы по организации мероприятий.


2. Права и обязанности пользователя

Использовать платформу только в законных целях
Не нарушать работу сервиса
Нести ответственность за достоверность введенных данных


3. Права и обязанности администрации

Предоставлять доступ к функционалу платформы
Обеспечивать сохранность данных в рамках localStorage
Улучшать сервис и исправлять ошибки


4. Ответственность

EventPro не несет ответственности за:
Действия подрядчиков и площадок
Результаты организации мероприятий
Потерю данных в случае очистки браузера пользователем


5. Изменение соглашения

Администрация оставляет за собой право изменять условия соглашения с уведомлением пользователей через платформу.
